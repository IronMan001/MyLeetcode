## 移动零

###Java版本

```Java

class Solution {
    public void moveZeroes(int[] nums) {
        
       
        for(int i=0;i<nums.length;i++){
           for(int j=0;j<(nums.length);j++){
               
                if((Math.abs(nums[i])>Math.abs(nums[j])) && nums[i]==0 || nums[j]==0 ){                
                    int temp = nums[i];
                    nums[i]=nums[j];
                    nums[j]=temp;          
                }
               
           }
 
        }
        
    }
    
}

```
